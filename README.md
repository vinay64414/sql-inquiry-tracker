# 📘 Customer Inquiry Tracker (SQL Project)

## 📌 Overview
This project demonstrates the design and implementation of a Customer Inquiry Tracking System using SQL. It focuses on managing customer support tickets through structured database operations and extracting meaningful insights from the data.

---

## 🎯 Objective
To apply core SQL concepts such as database creation, table design, CRUD operations, and data retrieval in a real-world support system scenario.

---

## 🧠 Key Concepts Used
- CREATE DATABASE & CREATE TABLE  
- INSERT INTO  
- SELECT with WHERE, LIKE, ORDER BY  
- UPDATE & DELETE operations  
- GROUP BY with COUNT  
- TIMESTAMP handling  

---

## ⚙️ Features
- 📥 Store customer inquiries with details  
- 🔍 Retrieve and filter inquiries (e.g., refunds, Gmail users)  
- 🔄 Update ticket status (Open, In Progress, Resolved)  
- 🗑️ Remove spam and outdated records  
- 📊 Analyze inquiries by issue type using aggregation
